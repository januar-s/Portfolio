Kreativschmidt.ch
==========
Dieses Projekt beinhaltet ein im Stile eines Drehbuchs gestaltetes Portfolio mit 3 Unterseiten. Dabei wurden viele kleine Animationen auf der Webseite versteckt.

Learnings
----
Im Zuge dieses Projektes habe ich mir GSAP angeeignet und damit einige Animationen umgesetzt. So zum Beispiel den Onscroll-Reveal des Textes auf der Startseite oder die Menüknöpfe.
Bei Erstem habe ich gelernt, dass damit die Animation smooth aussehen soll, sie möglichst kurz sein und im Bild erst Anfangen muss. Bei Zweitem lernte ich wie man einem Buttonhover ein gutes Feedback geben kann (manche sagen etwas zu viel aber naja)

Weiter habe ich bei "Mein Weg" gelernt, wie ich SVG's kontinuierlich Zeichnen lassen kann und Text manipulieren kann.
